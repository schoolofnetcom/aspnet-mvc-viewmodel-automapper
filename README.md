# ASP.NET MVC - View Models + AutoMapper
Projeto destinado aos desenvolvedores ASP.NET que queiram conhecer o MVC, com a utilização do Entity Framework para conectar e executar comandos de banco de dados, bem como manter a estrutura de banco de dados atualizada de acordo com a estrutura de código, com aplicação do Code First e Migrations.

Este projeto contém na sua estrutura a utilização de View Models para facilitar validações, formatos e outras propriedades necessárias nas views que não são correspondidas pelas classes de domínio.

E para facilitar o desenvolvimento e ganhar produtividade nas suas controllers, aplicou-se o AutoMapper para realizar o bind entre uma Model e uma View Model, de forma simples e objetiva.
